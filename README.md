## 📖 Sobre o projeto

O **Vintage Notes** nasceu da ideia de unir produtividade com conforto visual. Ao invés de um bloco de notas frio e genérico, a proposta foi criar algo com personalidade. 

O aplicativo funciona diretamente no **Android** utilizando **React Native + Expo**.

### 🎨 Estética do app
O visual e a interface cozy foram inspirados em:
* Cartas antigas
* Máquinas de escrever
* Papel parchment (pergaminho)
* *Journaling apps*
* *Dark academia*
* Minimalismo vintage

---

## 🚀 Funcionalidades

* 📝 Criar e editar notas de texto
* 📌 Salvar frases rápidas
* ☑️ Checklists interativas
* ⏰ Data e horário automáticos nas notas
* 🗑️ Excluir notas
* 💾 Salvamento local com `AsyncStorage`
* ☕ Aba “Buy Me a Coffee” integrada

---

## 🛠️ Tecnologias utilizadas

* **Framework:** React Native + Expo (Expo Router, Expo Font, Expo Clipboard)
* **Linguagem:** TypeScript e JavaScript
* **Banco de dados local:** AsyncStorage
* **Biblioteca utilitária:** React Native QRCode SVG

## ☕ Apoie o projeto ⋆☕︎♡

O app possui uma aba integrada de apoio aos criadores para simular uma experiência real de financiamento e ajudar a manter o projeto ativo e gratuito.

## 🎯 Objetivo

Mais do que um aplicativo de notas, o Vintage Notes busca criar uma sensação: um pequeno espaço confortável para pensamentos, listas, ideias e memórias.
